Detailed Description

A calendar.

The calendar provides navigation by month and year, and indicates the current day.

The calendar only allows selection of a single day, and you may listen for when the user selects a new date using the selected() signal.

The API is a subset of the WCalendar API:

  • limited to single selection mode: no setMultipleSelection(bool), select(const std::set<WDate>&) methods;
  • no support for programmatory navigation: no browseToNextMonth() , browseTo(), etc... methods
  • no support for getting information on what is currently displayed: no currentMonth() and currentYear() methods.
